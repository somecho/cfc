# Changelog

# 0.1.0

The first version of Soya. Current features:
- Drawing Api
  - 2D drawing API
    - `syDrawLine` 
    - `syDrawTriangle` 
    - `syDrawQuad` 
    - `syDrawPolygon` 
  - 3D drawing API
    - `syDrawUnindexed`
    - `syDrawIndexed`
  - Transformations
    - `syTranslate`
    - `syRotate`
  - Offscreen rendering with framebuffers `syFbo`
  - Custom shaders with `syBeginShader`
